
## Описание связей

- **Student** — хранит информацию о студентах
  - `id` — первичный ключ (автоматический)
  - `student_id` — номер студента
  - `full_name` — ФИО студента

- **Subject** — хранит названия предметов
  - `id` — первичный ключ
  - `name` — название предмета

- **Grade** — хранит оценки студентов по предметам
  - `id` — первичный ключ
  - `student_id` — внешний ключ на Student (один студент может иметь много оценок)
  - `subject_id` — внешний ключ на Subject (один предмет может иметь много оценок)
  - `value` — значение оценки (целое или дробное число)

## Типы связей

- Student → Grade: **один ко многим** (один студент — много оценок)
- Subject → Grade: **один ко многим** (один предмет — много оценок)
